# Whooing Desktop

[후잉(Whooing)](https://whooing.com) 웹 가계부를 데스크톱 앱으로 감싼 프로젝트입니다. [Tauri](https://tauri.app)를 사용해 macOS/Windows/Linux용 네이티브 데스크톱 배포판을 만듭니다.

## 다운로드

이 프로젝트는 아직 초기 개발 단계이며, 현재 배포되는 다운로드 빌드가 없습니다.

빌드가 준비되면 이 저장소의 [Releases](../../releases) 페이지에 macOS(.dmg) / Windows(.msi, .exe) / Linux(.deb, .rpm, .AppImage) 아티팩트가 게시될 예정입니다. Microsoft Store, Homebrew Cask, Snap Store 등록도 계획되어 있습니다.

그 전까지는 웹 브라우저에서 바로 이용할 수 있습니다.

👉 **https://whooing.com**

후잉은 이미 PWA로 제공되고 있어서, 브라우저에서 "홈 화면에 추가" 또는 "앱 설치"를 선택하면 데스크톱 앱과 유사하게 사용할 수 있습니다.

## 이 프로젝트에 대해

- 후잉 웹앱 자체의 기능을 새로 만드는 프로젝트가 아니라, 이미 완성된 웹앱을 여러 배포 채널(다이렉트 다운로드, 앱스토어 등)에 올리기 위한 래핑 프로젝트입니다.
- 모바일은 이 저장소의 스코프가 아닙니다. 안드로이드 앱은 별도로 이미 배포되어 있습니다.
